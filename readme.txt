Boletines
=========

Inicie el Programa ejecutando "Boletines.bat"

Si no funciona puede deberse a que no est‡ JAVA instalado en el equipo.
Si no es posible instalarlo, cree una carpeta al mismo nivel de 
BoletinesSinJava.bat llamada "jre" y copie una JAVA JRE dentro de modo
que exista jre\bin\java. Luego ejecute el programa usando BoletinesSinJava.bat
